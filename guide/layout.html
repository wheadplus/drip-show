# a-row 属性

参数 说明 类型 可选值 默认值
gutter 栅格的间隙，单位为 px String , Number —— ——
offset 举例左边的举例，单位为 px String , Number —— ——

# a-col 属性

参数 说明 类型 可选值 默认值
span,ipad,narrow,pc 不同屏幕宽度下分栏所占宽度的配置参数，键对应分栏所占宽度，值对应分栏前空隙宽度。span,ipad,narrow,pc,widePc对应 min-width 分别为 577px、769px、993px、1201px Object —— ——

# container 容器布局

容器布局，可快速搭建页面的基本结构，需浏览器支持flex布局。在使用容器布局时，container组件的子组件只能是header，content，footer，sider或container，同时，上述子组件的父组件也只能是container。

# 普通上中下三栏布局

点击查看代码 
<d-layout style="color: white; margin-bottom:50px;">
  <d-header class="demo-center" style="height: 50px; background:lightskyblue;">header</d-header>
  <d-content class="demo-center" style="height: 100px; background:deepskyblue;">content</d-content>
  <d-footer class="demo-center"  style="height: 50px; background:lightskyblue;">footer</d-footer>
</d-layout>
